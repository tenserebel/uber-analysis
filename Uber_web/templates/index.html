Description-:

This is exploratory data analysis(EDA) on Uber data.Exploratory data analysis is an approach of analyzing data sets to summarize their main characteristics, often using statistical graphics and other data visualization methods. In this project we explore the data to answer certain questions regarding the data as well as using dashboards to represent the data properly.

Problem Statement-:

Answering Questions using Eda and finding multiple details about the data.The questions being:

  • Relation between duration and purpose of the cab.
  • Which are the places having lowest starting point.
  • Which are the places having the highest starting point.
  • Miles most travelled.
  • Purpose of most cab.
  • Round trips.
  • How does the purpose of cab ride vary with time and distance.
  • Etc

Content of the dataset-:

The dataset includes data about rides in 2016 with total drives of 1155 and total miles of 12,204. The colums contains:

  • Start date
  • End date
  • Category
  • Start Time
  • Stop Time
  • Miles
  • Purpose

Notebook-:

The EDA is done in the notebook using python libraries such as pandas,numpy, matplotlib and seaborn. The EDA is used to answer questions in problem statement.

The data was imported using pandas. The data is then cleaned and altered according to the need. Then by using matplotlib and seaborn, the data is visualized and questions are answered.

Dashboard-:

The dashboard contains information of the data as well as visualization of the data using google data studio. The dashboards contains visualization about purpose of the cab, miles taken, starting stop and ending stop, Round trips, etc.

Conclusion-:

The conclusion of the whole project is:

  • Most of the cab rides are within a distance of 31 miles taking about 34 minutes.
  • Business Cab rides are not only more in volume, but also in distance travelled.
  • Main uses of cab rides are Meal/Entertainment, Customer visit, Meeting, Errand/Supplies.
  • Cab traffic is mostly concentrated in 5 cities or localities.
  • Ac seasonal pattern of cab ride volume exists, which is highest on December.
